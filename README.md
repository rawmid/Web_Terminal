# RTM Data Transfer Simulator 💻⚡

Simulates a hacker-style terminal in your browser with random logs, IPs, hashes, and a loading progress bar. Fun, aesthetic, and interactive.

**Features**

* Retro green-on-black terminal
* Randomized system logs & actions
* Dynamic progress bar with “encrypted packet” simulation
* Automatic finish messages with fallback

**Usage**
Open `index.html` in a browser and watch the terminal “transfer” data.

**Tech**: HTML, CSS, JavaScript

**Note**: Fully simulated – no real data is transferred.
